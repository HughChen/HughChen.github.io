Part 3: Antialiasing triangles

In this part, we implemented antialiasing. Deliverable - Additional structs and functions:

std::vector<unsigned char> superbuffer; void rasterize_point_super( float x, float y, Color color );

In order for antialiasing to work, we essentially ignore the framebuffer and use the supersample buffer with properly scaled up variables (vertices, indices, etc.). Then, we need to change the rasterize_point function, because it's expected behavior is to rasterize a point in the framebuffer. To make sure it keeps its behavior in the face of the downsampling, we simply set every superbuffer pixel that corresponds to the original framebuffer pixel to the value of the new point. Then, rasterize_point_super is the same as the previous rasterize_point function except it rasterizes in the superbuffer rather than the framebuffer. And the rasterize_triangle function calls rasterize_point_super over rasterize_point with it's scaled up variables.
